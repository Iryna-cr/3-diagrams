# 3-diagrams Завдання 6. Додаткове
## Тема - Онлайн-магазин

## Зміст:

### Проєкт містить три діаграми:

- Use Case - взаємодія користувача з системою через перегляд товарів, додавання до кошика та оформлення замовлення.

- Class Diagram - сутності системи: *User, Product, Cart, Order, Payment.*

- Sequence Diagram - процес оплати замовлення з участю користувача, веб-додатку та платіжного сервісу.

## Показує:
Як різні UML-діаграми описують поведінку, структуру та процеси однієї й тієї ж системи.
